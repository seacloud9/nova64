## 🚀 How to Use

### For Users
1. Open Nova64 in browser (`index.html`)
2. Click "📚 API Documentation" button in control panel
3. Browse categorized API cards on index page
4. Click any API to view documentation
5. Use "Back to Documentation Index" to return

### For Developers
1. Read the 4 fully documented APIs first (api, input, ui, audio)
2. Check stub pages for high-level overview of other systems
3. Refer to source code in `runtime/` for implementation details
4. Check `examples/` directory for usage examples
