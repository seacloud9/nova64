### Flight Controls

- **A/D or ←/→** - Yaw left/right (with auto-roll banking)
- **Q/E** - Strafe left/right
- **Shift** - Boost (high-speed mode)
- **Space** - Fire lasers (costs 5 energy)
- **M** - Fire missile (requires target lock)
- **T** - Target lock nearest enemy
- **Escape** - Pause game

### Combat Systems

#### Weapon Systems
- **Lasers**: Rapid-fire, energy-based, fired from ship wings
- **Missiles**: Limited ammo (10), homing when target locked
- **Target Lock**: Press T to lock onto nearest enemy in front

#### Enemy AI
- **Approach mode**: Closes distance when far away

## Technical Implementation

### Physics System
```javascript
// Ship movement with inertia
player.vel.x += forward.x * speed * dt;
player.vel.y += forward.y * speed * dt;
player.vel.z += forward.z * speed * dt;

// Drag for realistic space flight
player.vel.x *= 0.98;
player.vel.y *= 0.98;
player.vel.z *= 0.98;
```

### Missile Homing
```javascript
// Track locked target
if (proj.isMissile && proj.target && enemies.includes(proj.target)) {
  const toTarget = normalize(proj.target.pos - proj.pos);
  proj.vel += toTarget * CONFIG.MISSILE_TURN_RATE * dt;
}
```

### Camera System
```javascript
// First-person view follows ship orientation
camera.pos = player.pos;
camera.pitch = player.pitch;
camera.yaw = player.yaw;
camera.roll = player.roll;

// Camera shake on hit
if (camera.shake > 0) {
  camera.pos += random(-shake, shake);
}
```

## Fixed Issues

### Start Button Fix
- **Problem**: Game wouldn't start with Space/Enter
- **Root cause**: Used `isKeyPressed()` which has timing issues
- **Solution**: Changed to `isKeyDown()` for reliable detection
- **Result**: ✅ Space, Enter, and clicking all work

### Console.log Spam
- **Problem**: Console flooded with update messages
- **Solution**: Removed all console.log from game loops
- **Result**: ✅ Clean console output

### Game Over Restart
- **Problem**: Only R key could restart
- **Solution**: Added Space, Enter, and R for restart
- **Result**: ✅ Multiple restart methods
