## 🚀 How to Use

### For Users
1. Open Nova64 main page at `http://localhost:5173/`
2. Click the **📚 API Documentation** button in the control panel
3. Browse the categorized API list
4. Click any API to view detailed documentation
5. Use code examples to learn the API
6. Return to index with "← Back to Documentation Index" link

### For Developers
1. Access documentation at `http://localhost:5173/docs/index.html`
2. Search for the API you need (organized by category)
3. Read function signatures and parameters
4. Copy code examples to your game
5. Follow usage tips for best practices
6. Check related APIs for additional functionality
