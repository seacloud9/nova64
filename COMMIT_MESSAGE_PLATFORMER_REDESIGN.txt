## Technical Details

### Physics Implementation
```javascript
// Proper gravity application
player.vel.y -= CONFIG.GRAVITY * dt;

// Coyote time for forgiving jumps
if (player.grounded) {
  player.coyoteTime = 0.15;
} else {
  player.coyoteTime -= dt;
}

// Jump buffering for early presses
if (player.jumpBufferTime > 0 && player.coyoteTime > 0) {
  player.vel.y = CONFIG.JUMP_POWER;
  player.coyoteTime = 0;
  player.jumpBufferTime = 0;
}
```

### Level Design Philosophy
- Tutorial section (X: 0-20): Easy jumps to teach mechanics
- Easy section (X: 20-40): Build player confidence
- Medium section (X: 40-60): Introduce challenges and enemies
- Hard section (X: 60-80): Test player skills
- Victory platform (X: 80+): Reward completion

### Camera System
```javascript
// Side-scroller camera with smooth following
camera.target.x += (targetX - camera.target.x) * CONFIG.CAMERA_SMOOTH;
camera.pos.x = camera.target.x - 5;
camera.pos.y = camera.target.y + CONFIG.CAMERA_HEIGHT;
camera.pos.z = CONFIG.CAMERA_DISTANCE;
```

## Configuration
All game parameters now in centralized CONFIG object:
- PLAYER_SPEED: 8 (movement speed)
- JUMP_POWER: 12 (jump height)
- GRAVITY: 30 (gravity strength)
- CAMERA_DISTANCE: 15 (camera zoom)
- CAMERA_SMOOTH: 0.08 (camera follow smoothing)
